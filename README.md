# CellWall Google Tag Manager Template

Official Google Tag Manager (GTM) template for **CellWall**. This template provides a zero-code way to deploy the CellWall security and compliance engine to your website.

## Quick Links
- **[Installation Guide](https://cellwall.io/en/resources/blog/gtm-installation)** – Step-by-step setup instructions.
- **[CellWall Dashboard](https://platform.cellwall.io/)** – Get your Project ID here.
- **[Official Website](https://cellwall.io)** – Learn more about SiteWall and CookieWall.

## Features
- **SiteWall:** Real-time behavioral security monitoring and third-party script control.
- **CookieWall:** Automated, fully-compliant GDPR/CCPA cookie consent banner.
- **Zero-Code Installation:** Deploy the full platform via GTM without modifying your website's source code.

## Installation

### 1. Add Template
Search for **CellWall** in the [GTM Community Template Gallery](https://tagmanager.google.com/gallery/) and add it to your workspace.

### 2. Configure Tag
Create a new Tag and select the **CellWall** template.
 
### 3. Project ID
Enter your **Project ID**. You can find this in your [CellWall Dashboard](https://platform.cellwall.io/) under project settings.

### 4. Set Trigger
For the best protection and compliance, set the trigger to **Consent Initialization - All Pages** (or **All Pages** if consent initialization is not used).

## Security & Privacy
CellWall is built with a "Security First" architecture. Our engine interceptions happen at the browser level to protect your users from malicious scripts while ensuring a seamless compliance experience.
